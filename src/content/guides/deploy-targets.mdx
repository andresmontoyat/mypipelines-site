# Deploy targets

| Valor | Destino |
|-------|---------|
| `ec2` | SSH a un EC2 alcanzable directo |
| `ec2-vpn` | SSH a EC2 privado; el runner levanta un túnel WireGuard, despliega y lo baja |
| `eks` | Cluster EKS (manifests o Helm) |
| `s3` | Sitio estático (react/nginx) |
